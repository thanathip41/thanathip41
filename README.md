## 🚀 Tspace Ecosystem

> A collection of high-performance, TypeScript-first **npm packages** designed for building modern, scalable Node.js backend systems

---

### 📦 Packages

#### 🗄️ Database
- **[tspace-mysql](https://www.npmjs.com/package/tspace-mysql)** — tspace-mysql is an Object-Relational Mapping (ORM) tool designed to run seamlessly in Node.js and is fully compatible with TypeScript. It consistently supports the latest features in both TypeScript and JavaScript, providing additional functionalities to enhance your development experience.
 
 **Features**
  - ✅ Supports: MySQL / MariaDB / PostgreSQL  
  - 🧠 Type-safe by design  
  - ⚡ Lightweight & high performance  

---

### ⚡ Framework

- **[tspace-spear](https://www.npmjs.com/package/tspace-spear)**  
  A lightweight, high-performance API framework for Node.js, built on the native HTTP server with optional **uWebSockets.js (C++)** support for maximum speed and efficiency.

  **Features**
  - ⚡ Native HTTP performance  
  - 🔌 Optional WebSocket support (uWebSockets.js)  
  - 🚀 Designed for speed and scalability  

---

- **[tspace-0spear](https://www.npmjs.com/package/tspace-0spear)**  
  A lightweight and fast API framework for Node.js, focused on delivering the best **developer experience (DX)** while maintaining high performance.  
  Built on **uWebSockets (C++)** to provide a performant HTTP and WebSocket server.

  **Features**
  - 🧑‍💻 DX-focused design  
  - ⚡ Fast HTTP & WebSocket server  
  - 🧩 Easy to use and extend  

---

#### 🌐 Network

- **[tspace-nfs](https://www.npmjs.com/package/tspace-nfs)**

  **Features**
  - 🌐 Client & Server support  
  - 📁 Remote file access over network  
  - ⚡ Lightweight and efficient
